# About

Experimenting with page layouts.

# Requirements

- NodeJS >= 8.2.0
- A modern web browser

# Getting Started

- clone this repo
- `npm install`
- `npm run start`
- browse to `http://localhost:3000`

# Deployment

Zeit's [now](https://zeit.co/now) is used for deployment.

- `npm run deploy`

https://layout-experiment-zaxnfzmipx.now.sh/1
